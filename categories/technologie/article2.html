Sauvegarde de la base de données Mysql d'un site indexhibit

ouvrir un terminal Utiliser sudo tee pour rediriger la sortie avec les permissions root: sudo mysqldump -u "username" -p "nomdelabasededonnees par exemple : indexhibit2_db " | sudo tee /var/www/sauvegarde_indexhibit2_db_$(date +%Y%m%d_%H%M%S).sql > /dev/null

création de l'archive :

cd /var/www sudo tar -czvf sauvegarde_site_indexhibit_$(date +%Y%m%d_%H%M%S).tar.gz carnet

vérifier les sauvegardes

ls -lh /var/www/sauvegarde_site_indexhibit_*.tar.gz

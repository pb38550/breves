Plan de Construction d'un Site Web

  • Accueil
  • Technologie
  • Art

Pré-requis

  • Avoir un compte GitHub.
  • Installer Git sur votre machine.
  • Installer Visual Studio Code (VS Code).

Création du dépôt GitHub

  1. Créer un nouveau dépôt sur GitHub :
    • Connectez-vous à GitHub et créez un nouveau dépôt.
    • Nommer le dépôt, par exemple, mon-site-web.
    • Choisir de ne pas initialiser le dépôt avec un README pour commencer avec un dépôt vide.
  2. Cloner le dépôt sur votre machine locale : 
    git clone https://github.com/<votre-utilisateur>/mon-site-web.git
  3. Naviguer vers le répertoire cloné : 
    cd mon-site-web

Configuration du projet local

  1. Ouvrir le projet dans VS Code :
    • Ouvrir Visual Studio Code.
    • Cliquer sur File > Open Folder... et sélectionner le dossier mon-site-web.
  2. Créer la structure des fichiers : 
    .
mon-site-web/
├── index.html
├── styles.css
├── categories/
│   ├── technologie/
│   │   ├── index.html
│   │   ├── article-template.html
│   │   └── article1.html
│   └── art/
│       └── index.html
  3. Ajouter du contenu initial : 
    <!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mon Site Web</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <header>
        <h1>Bienvenue sur Mon Site Web</h1>
    </header>
    <div class="container">
        <nav class="sidebar">
            <ul>
                <li><a href="#">Accueil</a></li>
                <li><a href="categories/technologie/index.html">Technologie</a></li>
                <li><a href="categories/art/index.html">Art</a></li>
            </ul>
        </nav>
        <main class="content">
            <article>
                <h2>Introduction</h2>
                <p>Ceci est un exemple de contenu pour la page d'accueil.</p>
            </article>
        </main>
    </div>
</body>
</html>

Gestion des versions et déploiement

  1. Utiliser le terminal intégré de VS Code pour les commandes Git : 
    View > Terminal
  2. Ajouter les fichiers au suivi de Git : 
    git add .
  3. Committer les modifications : 
    git commit -m "Initial commit with project structure and basic content"
  4. Pousser les modifications vers GitHub : 
    git push origin main

Configuration de GitHub Pages

  1. Activer GitHub Pages dans les paramètres du dépôt :
    • Aller dans les Settings de votre dépôt sur GitHub.
    • Dans la section Pages, sélectionner la branche main et le dossier racine (/root) comme source.
    • Enregistrer les modifications.
  2. Accéder à votre site :
    • Après quelques minutes, votre site sera disponible à l'adresse https://<votre-utilisateur>.github.io/mon-site-web.

Ajouter de nouveaux articles

  1. Dupliquer article-template.html pour créer un nouvel article : 
    cp categories/technologie/article-template.html categories/technologie/article2.html
  2. Modifier le nouvel article :
    • Ouvrir et modifier article2.html dans VS Code pour ajouter le contenu réel.
  3. Mettre à jour index.html de la catégorie :
    • Ajouter un lien vers le nouvel article dans categories/technologie/index.html.
  4. Ajouter, committer et pousser les modifications : 
    git add .
git commit -m "Ajout de l'article technologie 2"
git push origin main
